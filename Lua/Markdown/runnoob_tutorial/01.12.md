#  1.12 模块与包

Lua的模块是由变量、函数等元素组成的table

```lua
-- 文件名 my_module.lua
-- 定义一个myModle的模块
MyModule ={}

MyModule.constant="Constant String"
MyModule.constant=120.00

function MyModule.func1()
    io.write("公共函数")
end
local function func2()
    io.write("私有函数")
end
function MyModule.func3()
    func2()
end

return MyModule
```

## require 函数

require函数用于加载模块

```lua
require("模块名")
require "模块名"
```
