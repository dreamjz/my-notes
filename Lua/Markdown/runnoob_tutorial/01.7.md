# 1.7 运算符

lua有以下运算符类型：

* 算术运算符
* 关系运算符
* 逻辑运算符
* 其他运算符

## 算术运算符

假设a=10，b=20

| 操作符 | 描述 | 示例        |
| ------ | ---- | ----------- |
| +      | 加法 | a+b --> 30  |
| -      | 减法 | a-b --> -10 |
| *      | 乘法 | a*b --> 200 |
| /      | 除法 | a/b --> 0.5 |
| %      | 取余 | a%b -->10   |
| ^      | 次幂 | a^2 -->100  |
| -      | 负号 | -a -> -10   |

```lua
a,b=21,10
--加法
print(a+b)
--减法
print(a-b)
--乘法
print(a*b)
--除法
print(a/b)
--取余
print(a%b)
--幂运算
print(a^2)
--负号
print(-a)
```

```lua
31
11
210
2.1
1
441.0
-21
```

## 关系运算符

假设a=10，b=20

| 操作符 | 描述     | 示例          |
| ------ | -------- | ------------- |
| ==     | 等于     | a==b -->false |
| ~=     | 不等于   | a~=b -->true  |
| >      | 大于     | a>b -->false  |
| <      | 小于     | a<b -->true   |
| >=     | 大于等于 | a>=b -->false |
| <=     | 小于等于 | a<=b -->true  |

```lua
--注意Lua的不等运算符为 ~=
a,b=1,2
if a~=b then
    print("a~=b")
end
```

```lua
a~=b
```

## 逻辑运算符

假设a=true，b=false

| 操作符 | 描述   | 示例             |
| ------ | ------ | ---------------- |
| and    | 逻辑与 | a and b -->false |
| or     | 逻辑或 | a or b -->true   |
| not    | 逻辑非 | not a -->false   |

## 其他运算符

| 操作符 | 描述                 | 示例             |
| ------ | -------------------- | ---------------- |
| ..     | 字符串连接           | "A".."B" -->"AB" |
| #      | 返回字符串或者表长度 | #“Hello” -->5    |

**注意**：#+string ，返回的是字符串的字节数

```lua
print(#"Hello") --5
print(#"你好") --6
print(#"こんにちは") --15
```

```lua
5
6
15
```

## 运算符优先级

```lua
--由高至低
^
not    - (unary)
*      /       %
+      -
..
<      >      <=     >=     ~=     ==
and
or
```

注意：-x^2 <--> -(x^2)
