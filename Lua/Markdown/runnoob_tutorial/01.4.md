# 1.4 循环控制

lua支持一下控制语句：

* break

* goto

## break

退出当前循环 ，若使用循环嵌套则退出最内层循环

### 示例

```lua
local a=0
while a<5 do
    print(a)
    a=a+1
    if a>3 then
        break
    end
end
```

```lua
0
1
2
3
```

## goto

goto将流程转至标记语句处

### 语法

```lua
goto Label
```

Label格式

```
:: Label ::
```

### 示例

```lua
local a=1
:: label1 :: print("---goto label---")
a=a+1
if a<3 then
    print(a)
    goto label1
end
```

```lua
---goto label---
2
---goto label---
```

使用goto可实现continue的功能

```lua
--输出奇数
for i=1,5 do
    if i%2==0 then
        goto continue
    end
    print(i)
    :: continue ::
end
```

```lua
1
3
5
```
