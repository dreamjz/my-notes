# 2.1 Redis命令

Redis命令用于在redis服务器上执行操作

### redis-cli

使用redis-cli启动redis客户端

```shell
redis-cli                            
127.0.0.1:6379> ping
PONG
```

ping指令用于检测redis服务是否启动

使用redis-cli在远程redis服务器上执行命令

```shell
$ redis-cli -h 127.0.0.1 -p 6379
127.0.0.1:6379> ping
PONG
127.0.0.1:6379> 
```
